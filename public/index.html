Import your Vagaro customers as Quo contacts. Two methods available.

Works Now

Method 1 — CSV Import

Export your customer list from Vagaro, then import it here. Covers all existing customers instantly, no API credentials needed.

  1. In Vagaro: Reports → Customer List Report → Export (choose CSV)
  2. Drop the file below or click to select it
  3. Review the preview, then click Import

Vagaro export columns: First Name, Last Name, Email, Cell Phone, etc.

Requires API Access

Method 2 — Live Webhook Sync

Vagaro pushes new and updated customers to this app automatically. Requires Vagaro API credentials (apply at vagaro.com/pro/updates/webhooks, up to 5-day approval).

In Vagaro: Settings → APIs & Webhooks → Add Webhook → paste this URL → select "Customers" events
